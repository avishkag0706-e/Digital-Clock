# 🕒 Digital Clock

A modern digital clock built using HTML, CSS, and JavaScript that displays real-time date and time with a clean UI and animated background.

---

## ✨ Features
- ⏱️ Real-time clock (updates every second)
- 📅 Displays current date
- 🎨 Modern UI with gradient animation
- 📱 Responsive and centered layout

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure
├── index.html
├── styleclock.css
└── clock.js


---

## 🚀 How to Run
1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Enjoy the live digital clock
